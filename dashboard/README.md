# AgentTrace Dashboard

React-based web dashboard for AgentTrace observability platform.

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

## Features

- Real-time trace visualization
- Cost tracking and analytics
- Waterfall view for agent execution
- Performance metrics

## Tech Stack

- Next.js 14
- React 18
- TanStack Query
- Recharts
- Tailwind CSS
